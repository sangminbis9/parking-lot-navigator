# 개인정보 / 권한 문서 템플릿

## 위치 권한 사용 목적

앱은 사용자가 선택한 목적지와 주변 주차장을 표시하고, 선택한 주차장까지 길안내를 제공하기 위해 위치 정보를 사용할 수 있습니다.

## 공유 확장 데이터 처리

사용자가 다른 앱에서 공유한 주소, 장소명, URL은 목적지 후보 생성에만 사용됩니다. 공유된 내용은 App Group 저장소에 임시 저장되며, 목적지 확인 후 앱 내부 기록에 저장될 수 있습니다.

## 외부 API 데이터

목적지 검색에는 Kakao Local API를 사용할 수 있습니다. 주차장 정보에는 서울 열린데이터광장 및 data.go.kr 데이터를 사용할 수 있습니다.

## 로그 수집 범위

운영 로그에는 요청 ID, provider 상태, 응답 품질, 오류 코드가 포함될 수 있습니다. 사용자의 민감한 원문 공유 텍스트나 API 키는 로그에 남기지 않습니다.
